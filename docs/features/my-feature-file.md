1. Feature Name: User Authentication
2. Goal: Provide users with a create account & login page that enables them to access private sections of the application
3. User Story: As a user, I want to create an account using my email and password. I want to be able to log-in and access private information about my account.
4. Functional Requirements:
Users can create an account with email and password
Users can log in to the application using email and pass
Logged in users can navigate to the profile page (private page)
Use Supabase Authentication
