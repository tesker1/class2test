A web application featuring user authentication and classic games.

Core features of your project from the user perspective:
- User Authentication: Create accounts, login, and access private profile pages using Supabase
- Tic Tac Toe Game: Play the classic 3x3 grid game with both single-player (vs AI) and two-player modes, featuring strategic computer opponent, win detection, and score tracking

Technology stack:
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Authentication: Supabase Authentication
- Hosting: Web browser (client-side application)
